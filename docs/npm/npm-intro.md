# npm 簡介


## npm scope 
定義npm package的命名空間(識別空間)並為package進一步分類，格式會是如下，存放形式的話會是以名為scope的目錄來存放名為package目錄

```
@scope/package 
```

例子：
```
@eslint/eslintrc
```

定義一個名為eslint的scope，並存放著名為eslintrc的package，同時在實體儲存空間會是以eslint目錄來存放eslintrc目錄，該eslintrc目錄存放eslintrc程式碼相關模組
