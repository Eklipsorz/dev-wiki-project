# Web API 風格
客戶端和伺服器端在網路上的溝通過程往往要考量幾個衡量點：
  - 客戶端和伺服器端平台如何傳遞/成功接收以及所需的資料形式是為何
  - 客戶端上的應用程式和伺服器上的應用程式如何傳遞/成功接收以及所需資料形式是為何
  - 客戶端和伺服器端之間的中介點如何幫忙成功轉遞以及所需的資料形式是為何
需要一個共同介面或者協議來讓彼此傳遞的內容都能轉換成彼此都能看得懂並處理得當的形式
